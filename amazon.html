About Amazon Affiliate Link Generator

Amazon Associates program is a great way to montetize your product recommendations across your blog, social media or youtube channel. In order for Amazon to track that you are referring visitors to amazon, you need to sign up for the Amazon Associates Program. Once you do it, you will be given an associates tag. This tag looks like mywebsite-20 and it is added to every link you want to be tracked on Amazon. The tool above helps you create the affiliate or referral link to Amazon.

How do i create an affiliate link to an Amazon product page?

An Amazon product links looks something like this: https://www.amazon.com/Backpack-Business-Compartment-Charging-Reflective/dp/B07MG4HN96/. You can create an affiliate link by adding ?tag=mywebsite-20 to the end of the link. There are a lot of cases when just adding your tag to the end of the URL is not sufficient and in this cases your tag will not be fired. For this purposes we’ve built the Amazon Affiliate Link Maker tool which helps you do this automatically without worrying about the edge-cases.

How does this link generator work?

We created Amazon Affiliate Link Generator to make it easier for you to create affiliate links to your favourite products on Amazon.
To start using this tool, simply find the product you like a lot on Amazon.For the purpose of this tutorial we are going to use this link to Amazon.
There are two main fields in this tool: Amazon Link URL and Affiliate Tag. You can find the affiliate tag you should use in your Associates Dashboard.
Log into your Dashboard to find the relevant affiliate tag for your country.
Keep in mind! The affiliate tag you use must correspond to the amazon store you are linking out to. For example: on Amazon.com you need to use your US affiliate tag.
There are multiple different affiliate dashboard for most of the countries where Amazon has its store.
